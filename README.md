# Actividad 4. Portafolio personal con Bootstrap

## Descripcion

Este proyecto corresponde a la Actividad 4 de la materia Programacion Web. La actividad consiste en desarrollar un portafolio personal utilizando tecnologias web, con el proposito de presentar de forma organizada las practicas realizadas durante el curso.

El sitio funciona como una pagina principal desde la cual se puede acceder a distintas tareas desarrolladas con HTML, CSS y JavaScript. Cada enlace dirige a un ejercicio independiente, permitiendo mostrar el avance en temas basicos de estructura web, estilos, formularios, manejo de eventos, funciones, arreglos, objetos y manipulacion dinamica de contenido.

## Objetivo

Crear un portafolio personal que recopile las actividades realizadas en la materia, aplicando conocimientos de diseno web, organizacion de archivos y navegacion entre paginas.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Contenido del proyecto

- `index.html`: pagina principal del portafolio.
- `tarea1.html` a `tarea18.html`: paginas correspondientes a las practicas realizadas.
- `css/`: carpeta que contiene los estilos del proyecto.
- `js/`: carpeta que contiene los scripts utilizados en las actividades.
- `img/`: carpeta destinada a imagenes usadas por el sitio.

## Actividades incluidas

El portafolio integra ejercicios como:

- Hola Mundo
- Manejo de texto, enlaces e imagenes
- Integracion de contenido
- Secciones y listas
- Horario y formulario
- Conversiones de unidades
- Verificacion de edad
- Uso de arreglos y objetos
- Calculadora basica
- Gestion de tareas
- Lista dinamica

## Instrucciones de uso

1. Abrir el archivo `index.html` en un navegador web.
2. Seleccionar cualquiera de las actividades listadas en la pagina principal.
3. Revisar la funcionalidad de cada ejercicio de manera individual.
